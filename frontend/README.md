# YouTube Automation - React Frontend

Modern React frontend for YouTube Automation app.

## Setup

```bash
cd frontend
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Tech Stack

- React 18 + TypeScript
- Vite (build tool)
- Tailwind CSS
- React Router
- React Query
- Lucide Icons

## Development

Frontend runs on `http://localhost:3000`
Backend API runs on `http://localhost:5001`

Vite proxy is configured to forward `/api/*` requests to the Flask backend.
