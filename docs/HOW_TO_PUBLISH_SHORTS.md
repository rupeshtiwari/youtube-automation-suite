- Wait for success message before closing modal
- If it happens: Delete duplicate from that platform directly

**Report:**
- Take screenshot showing both posts
- Note: Exact timestamp of both uploads
- Share in app logs or support

### Problem: Channel Connection Lost

**Fix:**
1. Go to **Settings** page
2. Find the platform showing ğŸ”´
3. Click "Reconnect" or "Connect"
4. Re-authenticate with that service
5. Should show ğŸŸ¢ **Connected** again

### Problem: "File Size Too Large"

**Solution:**
YouTube Shorts maximum: **128 MB**
- Compress video: Use HandBrake or FFmpeg
- Reduce resolution: 1080p instead of 4K
- Reduce frame rate: 24-30fps instead of 60fps
- Reduce audio bitrate: 128kbps instead of 320kbps

### Problem: Audio Not Working After Upload

**Check:**
1. Original file has audio (test in media player)
2. Audio codec is AAC or MP3
3. File isn't corrupted (re-download if needed)

**Action:**
- Try different video format (MP4 instead of WebM)
- Re-export with audio enabled
- Use pre-made template with audio included
